# Button

`Button` makes common actions immediately visible and easy to perform with one click or tap.

<Canvas>
  <Story name='Button'>{Template.bind({})}</Story>
</Canvas>

<ArgsTable story='Button' />

Buttons can have a responsive size based on the viewport width.

    {Template.bind({})}
  </Story>

If you need to access the `<button>` element directly, `Button` also has a `ref` prop that you can use to forward a ref to the underlying button.

<Canvas>
  <Story name='Forward refs'>{<ForwardRefs />}</Story>
</Canvas>

You can also use Styled Components to add custom styles to `Button`, e.g. `styled(Button)`.

`Button` supports optional `IconLeft` and `IconRight` props to display optional iconography.

These props expect components from `pcln-icons`.
